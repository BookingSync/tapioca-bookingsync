# Tapioca bookingsync

## Installation
```
pip install tapioca-bookingsync
```

## Documentation
``` python
from tapioca_bookingsync import BookingSync


api = BookingSync(
	client_id='{your-client-id}', access_token='{any-valid-access-token}')

```
